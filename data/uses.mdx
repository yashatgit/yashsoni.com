# Tools I use

- [QLMarkdown](https://github.com/sbarex/QLMarkdown) - Preview Markdown files on Mac Finder
- [Monitor Control](https://github.com/MonitorControl/MonitorControl) - Monitor Brightness control
- [HEIC Convertor](https://imazing.com/heic/download) - iOS image conversions made super easy!

## Terminal
